윈도우에서 터미널 키는법:윈도우키->스페이스바->cmd입력


dir  : 현재 폴더 안에있는 모든 파일과 폴더들을 보는역할(명령어, 커멘드)
directory(=folder)의 약자\
/slash
\back slash

cd 경로(폴더)이름 : 현재 폴더 안에 있는 다른 폴더로 이동.
cd .. : 현재 폴더의 상위(부모) 폴더로 이동.
경로를 한꺼번에 표현하려면, 처음경로/다음경로/그다음경로/그다음다음경로.. 이런식으로 표현한다.
cd = change directory

mkdir : 현재 폴더 안에 새 폴더를 만든다.
mkdir = make directory
